First make sure you've prepared MSYS2 environment
by reading the "MSYS2-prepare.md" found in readme folder

update contents of config files as needed.

open VS cmd prompt (x64 or x86 Visual Studio Native cmd prompt depending on build type)

for x64 cmd prompt copy-paste following and hit enter:
D:\Github\Visual-Studio-gtkmm\projects\compile-icons\msvc-140\start-64

for x86 cmd prompt copy-paste following and hit enter:
D:\Github\Visual-Studio-gtkmm\projects\compile-icons\msvc-140\start-32

copy-paste one of the following lines (depending on your build type) to MSYS2 shell and hit enter:

/d/Github/Visual-Studio-gtkmm/projects/compile-icons/msvc-140/release-32.sh
/d/Github/Visual-Studio-gtkmm/projects/compile-icons/msvc-140/release-64.sh
/d/Github/Visual-Studio-gtkmm/projects/compile-icons/msvc-140/debug-32.sh
/d/Github/Visual-Studio-gtkmm/projects/compile-icons/msvc-140/debug-64.sh
